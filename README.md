# TrainScheduler
Basic Train Scheduler (Firebase and Moment.js)

![Alt text](assets/images/trainscreenshot.jpg?raw=true "Trivia Screenshot")


- How does it work? 
  * Train Schedules are displayed in the table.
  * Users can add a new Train/Destination/Frequency/Start Time.
  * New Schedule is Appended to the table.
  * All schedules are saved in Firebase and will save on page resets

- What is the goal of this project?
  * To demonstrate basic understanding of Firebase Database for persistance and moment.js to handle the time calculations.
